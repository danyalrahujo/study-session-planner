**Study Session Planner** is a desktop application designed to help students organize and manage their study activities through a simple graphical user interface. Users can create new study sessions by providing a unique ID and a description. The application allows users to view, update, and delete study sessions at any time.

In addition, the application supports tag management, enabling users to create, update, and remove tags that can be used to categorize study activities.
